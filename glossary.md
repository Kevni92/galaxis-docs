# Glossar

Dieses Glossar definiert die verbindlichen Grundbegriffe von Galaxis. Fachliche Dokumente müssen diese Begriffe konsistent verwenden.

| Begriff | Definition |
|---|---|
| **AI** | Automatisierte Steuerungslogik, die über einen AI-Controller ein Reich steuert. |
| **Befehl** | Fachlich definierte Anweisung eines Controllers an ein kontrolliertes Spielobjekt oder Reich. |
| **Controller** | Instanz, die Entscheidungen für ein Reich trifft. Ein Controller kann ein Spieler oder eine AI sein. |
| **Ereignis** | Durch die Simulation ausgelöste Situation, die Informationen, Auswirkungen oder Entscheidungen erzeugt. |
| **Flotte** | Organisierte Gruppe aus einem oder mehreren Schiffen, die gemeinsam Befehle ausführt. |
| **Galaxie** | Gesamter räumlicher und politischer Spielraum einer Kampagne. |
| **Galaxieansicht** | Räumliche Clientansicht bekannter Sternensysteme und Verbindungen. Die Darstellung kann 3D sein; fachliche Erreichbarkeit folgt dem Systemgraphen. |
| **Himmelskörper** | Sammelbegriff für Sterne, Planeten, Monde und weitere natürliche Objekte in einem Sternensystem. |
| **Kampagne** | Eine persistente Partie von ihrer Erzeugung bis zu ihrem Abschluss oder Abbruch. |
| **Kolonie** | Dauerhaft bewohnter und von einem Reich kontrollierter Standort auf einem kolonisierten Himmelskörper. |
| **Kontextaktion** | Für eine aktuelle Auswahl und ein Ziel serverautoritativ angebotene oder bestätigte mögliche Aktion, die zu einem fachlichen Befehl führen kann. |
| **Lokale XY-Ebene** | Autoritative zweidimensionale Geometrie innerhalb eines Sternensystems für Position, Bewegung, Reichweite, Begegnung und Kampf. |
| **Planet** | Himmelskörper, der abhängig von seinen Eigenschaften erforscht, genutzt oder kolonisiert werden kann. |
| **Reich** | Politische und wirtschaftliche Organisation, die Gebiete, Kolonien, Flotten und weitere Spielobjekte kontrolliert. |
| **Ressource** | Messbare Größe, die erzeugt, gelagert, verbraucht oder gehandelt werden kann. |
| **Schiff** | Einzelne mobile Raumeinheit mit definierten Eigenschaften und Fähigkeiten. |
| **Spieler** | Menschlicher Teilnehmer, der über einen Spieler-Controller ein Reich steuern kann. |
| **Sternensystem** | Räumlich zusammengehöriger Bereich um mindestens einen Stern mit Himmelskörpern und möglichen Verbindungen zu anderen Sternensystemen. |
| **Sternensystemansicht** | Dreidimensional gerenderte Clientansicht eines bekannten Sternensystems. Die fachliche lokale Geometrie bleibt zweidimensional. |
| **Vertrag** | Verbindliche fachliche Beschreibung von Befehlen, Zuständen oder Schnittstellenanforderungen zwischen Systemteilen. |

## Pflege

Neue zentrale Begriffe müssen ergänzt werden, sobald sie in mehreren Dokumenten verwendet werden oder eine eindeutige Abgrenzung benötigen.

Der ausführliche, lokalisierbare Begriffskatalog für Tooltips und rekursive Popovers wird aus diesem Glossar und den Fachdokumenten abgeleitet. Er steht unter dem UI-/UX-Konzept und darf keine abweichenden Definitionen einführen.
