# Designentscheidungen

Dieser Ordner enthält dauerhaft nachvollziehbare fachliche Designentscheidungen.

## Benennung

```text
NNNN-kurzer-titel.md
```

Beispiel: `0001-serverautoritative-simulation.md`

## Status

- vorgeschlagen,
- angenommen,
- verworfen,
- ersetzt.

## Regeln

- Jede Entscheidung verwendet die [Entscheidungsvorlage](../templates/decision-template.md).
- Angenommene Entscheidungen sind gegenüber allgemeinen Fachdokumenten vorrangig.
- Ersetzte Entscheidungen müssen auf ihren Nachfolger verweisen.
- Eine Entscheidung beschreibt Kontext und Folgen, nicht nur das Ergebnis.
