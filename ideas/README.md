# Ideen

Dieser Ordner enthält unverbindliche Überlegungen für mögliche spätere Erweiterungen von **Galaxis**.

> **Wichtig:** Inhalte unter `ideas/` gehören ausdrücklich **nicht** zur fachlichen Dokumentation. Sie sind weder beschlossen noch freigegeben und dürfen keine Spielregeln, REST-Verträge, Implementierungen, Tests oder Architekturentscheidungen beeinflussen.

Für Agenten gilt:

- Ideen dürfen nur berücksichtigt werden, wenn eine Aufgabe ausdrücklich auf die jeweilige Idee verweist.
- Eine Idee darf nicht als Anforderung, Akzeptanzkriterium oder geplantes Feature behandelt werden.
- Aus einer Idee dürfen nicht automatisch Client-, Server- oder Docs-Issues abgeleitet werden.
- Vor einer Umsetzung muss die Idee im regulären Workflow fachlich bewertet, entschieden und in die verbindliche Dokumentation überführt werden.
- Bei Widersprüchen sind Ideen immer zu ignorieren.
