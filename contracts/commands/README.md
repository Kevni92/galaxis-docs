# Befehlsverträge

Hier werden freigegebene Befehle beschrieben, die ein Spieler- oder AI-Controller erteilen kann.

Jeder Befehl muss mindestens enthalten:

- fachlichen Namen,
- ausführenden Akteur,
- Zielobjekt,
- Parameter,
- Voraussetzungen und Berechtigungen,
- Validierungsregeln,
- Kosten und Dauer,
- Auswirkungen,
- Fehlergründe,
- Rückmeldung an den Controller.

Verwende die [Befehlsvorlage](../../templates/command-template.md).
