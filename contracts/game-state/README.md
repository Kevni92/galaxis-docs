# Verträge zum Spielzustand

Hier wird beschrieben, welche fachlichen Zustände Client, Server oder andere Teilnehmer austauschen und interpretieren müssen.

Zu dokumentieren sind insbesondere:

- Bedeutung jedes Zustands,
- Sichtbarkeit und Berechtigung,
- erlaubte Werte,
- Beziehungen zwischen Zuständen,
- zeitliche Gültigkeit,
- vollständige und teilweise Ansichten,
- Umgang mit unbekannten oder verborgenen Informationen.
